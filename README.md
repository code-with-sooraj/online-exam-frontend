# Online Examination Portal - Frontend (Vite + React)

## Scripts
- dev: Start dev server
- build: Production build
- preview: Preview built app

## Setup
1. Install deps
```
npm install
```
2. Start dev
```
npm run dev
```
3. Configure backend URL by setting `VITE_API_URL` in `.env`.

## Notes
- Auth is mocked. Replace with real backend endpoints.
- Pages included: Login, Student Dashboard, Exam (with timer), Admin Dashboard, Create Exam.
- UI: TailwindCSS.
